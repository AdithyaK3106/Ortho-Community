# Roadmap

What we're building and when.

---

## Phase 1: Foundation (Now)

**Status:** Shipped

What's working:
- Repository intelligence: Python indexing (symbols, imports, call graph)
- Architecture intelligence: detection across 5 styles, subsystems, cycles
- Context assembly: full-text search, impact analysis
- Local-first storage (single SQLite file)
- CLI: `init`, `scan`, `analyze`, `context`, `status`

---
